# Selenium Java Automation Framework

## Project Overview

This project demonstrates a Selenium Test Automation Framework developed using Java, Selenium WebDriver, TestNG, and Maven.

The framework follows the Page Object Model (POM) design pattern to improve maintainability, reusability, and scalability of automated test scripts.

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* Git
* GitHub

## Framework Design

* Page Object Model (POM)
* Base Test Class
* Utility Classes
* TestNG Framework

## Project Structure

src/test/java

* pages
* tests
* utils
* base

## Test Scenarios

* Login Validation
* Logout Validation
* Invalid Login Validation

## How to Run

1. Clone the repository
2. Open in IntelliJ IDEA or Eclipse
3. Install Maven dependencies
4. Execute testng.xml
